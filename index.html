HTML Notes

HTML elements consist of opening and closing tags and the content.

Tag Name Description
<h1> Header 1 The most important/bigger heading. Headings go from h1 to h6.
<body> Body the body of the HTML document.
<head> Head A container for the meta data about the HTML. This can include links to the stylesheet.
<title> Title Defines the title of the HTML. This what people will see in their tab heading.
<p> Paragraph Defines a paragraph.
<ul> Unordered List An unordered bulleted list.<li> is used to create the list items.

CSS Notes

CSS is used to personalize html. Things like background color, font color, font family and other attributes can be made unique to the userss desire.

Attribute Definition
color Changes the color of the specified selector
font-family Changes the font style
background-color Changes the background color
margin Used to create space around elements, outside of any defined borders
padding Used to create space around content, inside of any defined borders
